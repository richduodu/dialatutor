# **App Name**: Voice2Learn Proof

## Core Features:

- Voice Lesson Interface (IVR): Allows students to dial a phone number and interact with an automated voice system to take oral lessons and provide responses.
- Speech-to-Text Transcription: Transcribes student's oral responses from the IVR system into text for further processing.
- AI-Powered Lesson Evaluation Tool: Utilizes AI to evaluate the transcribed student responses against lesson criteria, determining completion and comprehension.
- Proof of Learning Minting: Automatically mints a unique immutable Proof of Learning token (ERC-721/Soulbound Token) on the Polygon PoS blockchain upon successful lesson completion.
- SMS Learning Confirmation: Sends an SMS notification to the student with a confirmation of their achievement, including a link to the blockchain transaction on a block explorer.
- Funder Verification Dashboard: A simple user interface allowing funders to view minted Proof of Learning tokens and easily access links to block explorer transactions for verification.

## Style Guidelines:

- Primary brand color (learning and trust): Blue, hex value '#5A8EDB'. This calm and authoritative hue fosters a sense of learning and reliability, providing a professional yet approachable feel.
- Background color: Light blue-grey, hex value '#EFF3F8'. This soft, almost neutral backdrop creates a clean, uncluttered visual space that enhances content readability.
- Accent color (progress and success): Cyan, hex value '#15AFC4'. This vibrant, analogous color provides clear visual cues for important actions and positive feedback, ensuring key elements stand out.
- Both headlines and body text will use 'Inter', a modern grotesque sans-serif font known for its exceptional readability and clean, objective appearance across various text sizes.
- Utilize simple, clean line-art icons that visually communicate learning, verification, and blockchain concepts, ensuring universal understanding and maintaining a professional aesthetic.
- Implement a clean, card-based layout for the web dashboard, prioritizing responsive design to ensure optimal viewing and interaction across devices, especially for verifying transactions on mobile.
- Incorporate subtle feedback animations for actions such as lesson completion or successful token minting, using understated transitions to provide clarity without distraction.
